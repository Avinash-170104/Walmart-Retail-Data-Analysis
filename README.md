# Walmart-Retail-Data-Analysis
## Walmart Retail Data Analysis – Tableau Dashboard

## 📌 Project Overview
This project presents an interactive Tableau dashboard analyzing Walmart retail sales performance. The dashboard provides insights into sales, profit, discounts, and regional performance to help understand business trends and profitability.
The main objective of this project is to demonstrate data visualization and business intelligence skills using Tableau.

## 📈 Key Metrics
The dashboard highlights important business KPIs:
Average Sales: $1,775.88
Average Profit: $181.18
Average Discount: $0.05
These KPIs help evaluate overall retail performance and profitability.

## 📊 Dashboard Insights
🗺 Average Profit by State
A geographical visualization showing the average profit distribution across different states, helping identify high-performing and low-performing regions.

## 👥 Age vs Profit
This chart analyzes the relationship between customer age groups and profit generation, providing insights into which age segments contribute most to revenue.

## 📅 Regional Quarterly Average Profits
A heatmap that displays quarterly profit trends by region from 2012–2015, allowing easy identification of seasonal trends and regional business performance.
